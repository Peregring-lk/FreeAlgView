README:
        FreeAlgView
    ===================

FreeAlgView is a libre and learning-oriented algorithm animation
application. Mainly, this package contains the source code of the FreeAlgView
application. No object code is distributed in this package.

You can get more information about the project, its evolution and other links
related to this project and application in the FreeAlgView blog:

   - http://freealgview.blogspot.com

You can get this package in zip and tar.gz formats from the next url:

   - https://github.com/Peregring-lk/FreeAlgView/downloads
